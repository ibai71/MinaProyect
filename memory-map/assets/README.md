# Memory Map assets

This folder contains the image conventions for the city memory map.

## Base map

The page uses a local base map file:

- `assets/world-map.svg`

## Pin images

Add mini city images in `assets/pins/` with these filenames:

- `seoul.png`
- `busan.png`
- `bangkok.png`
- `hanoi.png`
- `kuala-lumpur.png`
- `da-nang.png`
- `saigon.png`
- `madrid.png`
- `hong-kong.png`
- `tokio.png`

## Memory images

Add memory photos in `assets/memories/` with these filenames:

- `seoul.png`
- `busan.png`
- `bangkok.png`
- `hanoi.png`
- `kuala-lumpur.png`
- `da-nang.png`
- `saigon.png`

Pending cities (`Madrid`, `Hong Kong`, `Tokio`) do not require a memory image yet.
